#  运算

1. 算数运算
```
 + - * / % ++ --

 +  算数运算   字符串的连接
    定理： 任何数据类型和字符串相加，最终的结果都是字符串
    python稍有不同

    ++ 前后的问题

    如果说有多条命令在统一行，++在前面  先执行++，否则 先执行其他命令
```

2. 逻辑运算符

```
  或  与   非
  ||  &&   !
  or  and  not
```

3. 比较运算
```
  > <  <= >= == ===  != !==

  =  代表赋值
  ==  比较 (只比较值)
  === 完全比较 (不但比较值，数据的类型)
```
4. 赋值运算
```
+
+=
-=
*=
```
